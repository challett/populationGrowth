This program changes the desired image half way through the number of specified generations.

compile with
make acc

run it with
./genimg_acc <firstimage.ppm> <outimage.ppm> <numgens> <popsize> <secondimage.ppm>

example of output images into video:
https://www.youtube.com/watch?v=YKD05A3p_-I&feature=youtu.be

output images can be found in ./output
